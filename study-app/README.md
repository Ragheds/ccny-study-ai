# CCNY Study AI

CCNY Study AI is an AI-powered study assistant for City College students. It is designed to help students choose their major, find relevant courses, organize study material, and eventually generate quizzes, flashcards, notes, and study plans.

## Current Status

The project is in active development. The first version focuses on building the core student flow:

- Browse CCNY majors
- Open a major dashboard
- Search and select courses from the catalog
- Prepare the app for saved courses and AI study tools

## Roadmap

### Phase 1: Majors

Status: Done

- Display CCNY schools and majors
- Link each major to a dashboard page

### Phase 2: Courses

Status: In progress

- Show courses grouped by school and department
- Search courses by code or name
- Select courses for a student plan

### Phase 3: Student Profile

Status: Planned

- Save selected major
- Save selected courses
- Build a student study profile

### Phase 4: AI Workspace

Status: Planned

- Chat-based tutoring
- Quiz generation
- Flashcards
- Notes

### Phase 5: File Uploads

Status: Planned

- Upload syllabi
- Upload lecture PDFs
- Upload notes

### Phase 6: Study Analytics

Status: Planned

- Track quiz scores
- Track study streaks
- Show progress over time

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.
